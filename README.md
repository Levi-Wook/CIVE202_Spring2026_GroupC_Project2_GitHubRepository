GitHub Repository for CIVE 202 - Group C- Project 2

Client: Nebraska Department of Transportation (NDOT)

Project Summary
The Nebraska Department of Transportation (NDOT) has contracted our firm to migrate its existing Excel-based concrete mix design model into a Python-based automated system. The objective of this project is to replicate the calculation logic contained in the NDOT “Mix Design” worksheet and implement it in a structured, modular, and repeatable Python workflow.

This automation will replace manual Excel calculations with Python functions and user input prompts. By transitioning to Python, the mix design process becomes more transparent, consistent, and suitable for future integration.

The program calculates and generates a formatted weight chart for one cubic yard of concrete, based on user-specified material properties and design parameters.
